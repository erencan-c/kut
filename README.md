# Kut
The main Kut Language Interpreter repository (Kut Dili Yorumlayıcısı ana deposu)

For more information, refer to the [wiki](https://github.com/erencan-c/kut/wiki)

# Examples

### Hello World

```(ekran "Merhaba, Dünya" yazsın)```

### Variable Creation

```('değişken 5 olsun)```

### Basic Arithmetic

```(((((5 2 +) 3 *) 10 -) 2 /) 6 ^)```

### Combined

```
('x 10 olsun)
('y 8 olsun)
(ekran (x y +) yazsın)
```
